# DragableToggle

Clean draggable square toggle button with smooth rounded corners (iOS style).

## Features
- Square shape with smooth rounded corners
- Fully draggable
- Tap to toggle ON/OFF with color change
- Smooth animations
- Clean modern look

## How to use
1. Build with Theos
2. Inject the resulting `.dylib` into any app
3. A draggable toggle button will appear on screen

Made for easy testing and injection.
